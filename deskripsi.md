# My Profile Web

Website ini merupakan project personal berbasis HTML, CSS, dan JavaScript yang digunakan untuk menampilkan informasi profil diri secara terstruktur dan menarik. Konten yang disajikan meliputi biodata, family, hobi, cita-cita, daily activities, serta gallery yang berisi gambar dan musik. Selain itu, website ini juga dilengkapi dengan elemen interaktif sederhana untuk meningkatkan pengalaman pengguna.

Project ini dibangun menggunakan HTML sebagai struktur utama halaman, CSS untuk pengaturan tampilan dan desain, serta JavaScript untuk menambahkan interaksi. Tampilan website menggunakan font Poppins agar terlihat modern dan rapi.

Struktur folder pada project ini terdiri dari file utama seperti index.html, style.css, dan script.js, serta folder assets yang berisi gambar dan file musik yang digunakan dalam website. Susunan ini dibuat agar project lebih terorganisir dan mudah dikembangkan.

Untuk menjalankan website secara lokal, pengguna dapat membuka folder project menggunakan Visual Studio Code, kemudian menjalankan file index.html menggunakan Live Server. Website ini juga dapat dihosting agar dapat diakses secara online menggunakan layanan seperti GitHub Pages atau Netlify.

Tujuan pembuatan website ini adalah untuk memenuhi tugas pembuatan web sekaligus melatih pemahaman dasar dalam pengembangan website, khususnya dalam penggunaan HTML, CSS, dan JavaScript. Ke depannya, website ini masih dapat dikembangkan lebih lanjut dengan menambahkan fitur seperti animasi, tampilan responsif, serta peningkatan interaktivitas.
